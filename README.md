# Ketan Catholic Junior High School Website

## Description
This is the official website for Ketan Catholic Junior High School, located in Takoradi, Ghana. The website showcases the school's mission, vision, academics, co-curricular activities, admissions process, and more. It emphasizes excellence, discipline, and faith in education.

## Features
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Navigation**: Hamburger menu for mobile, dropdown navigation for desktop.
- **Hero Section**: Image slider with overlay text.
- **Multiple Pages**: Includes sections for about, academics, admissions, gallery, contact, and more.
- **Media Integration**: Videos and images for co-curricular activities, spiritual guidance, and school events.
- **Contact Information**: Direct links to email, phone, and social media.

## Technologies Used
- **HTML5**: Structure of the website.
- **CSS3**: Styling and responsive design.
- **JavaScript**: Interactive elements like menu toggle and date/time display.
- **Node.js**: For serving the static site locally (via `serve` package).

## Installation and Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ketan-catholic-jhs.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ketan-catholic-jhs
   ```
3. Install dependencies (if needed for local serving):
   ```bash
   npm install
   ```
4. Start the local server:
   ```bash
   npm start
   ```
   This will serve the site at `http://localhost:3000` or similar.

Alternatively, open `index.html` directly in a web browser for a static view.

## Usage
- Browse the website by navigating through the menu items.
- For admissions, fill out the application form and follow the checklist.
- View gallery images and videos in the respective sections.
